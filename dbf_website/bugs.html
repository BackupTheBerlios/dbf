Bugs & Feature Requests

To report Bugs for libdbf and dbf, please have a look at the development team's pages at Berlios. You can reach the tracker for all actual posted bugs by following the link below:

http://developer.berlios.de/bugs/?group_id=1219



It is also possible to send us wishes for upcoming versions of libdbf and dbf. Please use the Feature Request Tracker to post your wishes:

http://developer.berlios.de/feature/?group_id=1219



If you need special help, please visit the forum or post a support request on the development team's page. The following links might be helpful:

Support Request: http://developer.berlios.de/support/?group_id=1219
Forum: http://developer.berlios.de/forum/?group_id=1219
